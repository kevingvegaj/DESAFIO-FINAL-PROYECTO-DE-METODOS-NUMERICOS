Descripción del Proyecto

Este proyecto implementa diferentes métodos numéricos para modelar escenarios reales relacionados con abastecimiento de combustibles, precios de alimentos, pérdida del poder adquisitivo y análisis de estabilidad.

Módulo A - Sistemas de Ecuaciones Lineales

Métodos: Jacobi, Gauss-Seidel y LU.

Módulo B - Ecuaciones Diferenciales

Métodos: Euler, Heun y Runge-Kutta 4.

Módulo C - Interpolación

Métodos: Lagrange, Newton y Splines.

Módulo D - Integración Numérica

Métodos: Trapecio, Simpson 1/3 y Simpson 3/8.

Módulo E - Raíces de Ecuaciones

Métodos: Bisección, Newton-Raphson y Secante.
